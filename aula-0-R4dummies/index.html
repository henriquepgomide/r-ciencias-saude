R para iniciantes

  • No curso usaremos o RStudio e o RCommander
  • RStudio - Deixar sua vida mais fácil para gerenciar pacotes, salvar gráficos e fazer análises
  • RCommanderr - Apontar e clicar para rodas as análises estatísticas
  • Diversos tutoriais e formas de analisar dados serão divulgados para facilitar sua vida.

Como instalar

  • R - Instale o R aqui.
  • Rstudio - Clique no botão "Download Rstudio Desktop" neste link
  • Rcommander - Abra o Rstudio, vá em "tools > Install Packages" e digite "Rcmdr", Dê ok. Clique em instalar
  • Vídeo de instalação no youtube.
